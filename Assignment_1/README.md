# 📘 Statistical Metods in A.I. Assignment-1

## 📌 Overview

This repository contains the implementation and analysis for **Assignment 1 (Spring 2026)** of the *Statistical Methods in Artificial Intelligence* course.

The assignment consists of four questions covering:

* Bias-Variance / Subspace Methods / Clustering (Q1 – one variant)
* Data Normalisation and KNN Classification (Q2)
* Linear Regression under Noise and Outliers (Q3)
* Concept Deep-Dive (Q4)

All implementations are done **from scratch using only permitted libraries** (NumPy, Matplotlib, Pandas), ensuring full compliance with assignment constraints.
